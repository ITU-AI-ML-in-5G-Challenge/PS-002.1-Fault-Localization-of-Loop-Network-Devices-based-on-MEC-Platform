# 基于MEC平台的网络设备故障定因

根据提供的训练数据来进行算法模型训练，并使用模型对故障测试数据进行分析，快速预测故障趋势或定位故障根因并复现故障发生过程。

## 使用方式

1. 命令行执行

   ```
   python main.py
   ```

   直接运行脚本产生输出文件

2. 输出

   产生模型文件，节点得分文件以及结果文件。

   分别输出在：

   ```
   model/    # 模型文件
   output/process/    # 节点得分
   output/result/    # 结果
   ```
